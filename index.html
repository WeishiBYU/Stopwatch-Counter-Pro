Installation Instructions

  1. Download the extension zip file using the button below.
  2. Unzip the downloaded file to a folder on your computer.
  3. Open Chrome and navigate to chrome://extensions.
  4. Enable "Developer mode" by toggling the switch in the top-right corner.
  5. Click "Load unpacked" and select the folder where you unzipped the extension.
  6. The Stopwatch Counter Pro extension should now appear in your browser toolbar!

How to Use

Stopwatch

Click the "Start" button to begin timing. The stopwatch will continue to run even if you close the popup. Click "Stop" to pause timing and "Reset" to clear the stopwatch.

Creating Counters

Enter a name in the "Enter counter name" field and click "Add Counter." Use the + and - buttons to increment or decrement the counter. The trash icon removes a counter.

Data Visualization

Your counter values are automatically visualized in the chart area. Use the dropdown to switch between different chart types.

Comparing Data

Click the "Compare" button to see a comparison chart that shows counter values versus time.

Exporting Data

Click the "Export to CSV" button to download your counter data in CSV format.

Session Management

Use "Save Session" to store your current stopwatch time and counter data. "View History" allows you to see and load previous sessions.
